# 📘 SSAFY 오프라인 알고리즘 스터디

알고리즘 풀이를 체계적으로 기록하고 관리하기 위한 스터디 저장소입니다.
<br>
<br>

## ✅ 사용 방법
1. 본인 이름의 폴더를 생성한다.
2. 생성된 폴더 안에 **월별 폴더**를 생성합니다.
3. 알고리즘 풀이 후 **날짜에 맞게 파일을 업로드**합니다.
<br>

## 📁 Repository 폴더 구조
    {이름}/{월}/{날짜}/{코드}
- 💡 예시: `김유준 / 2월 / 0212 / 코드`
<br>

## ❗ 코드 제목 양식
    {플랫폼}_{난이도}문제번호_문제제목.언어 - {기타 내용}
- 💡 예시: `BOJ_[실4]1219_문제제목.py`
<br>
